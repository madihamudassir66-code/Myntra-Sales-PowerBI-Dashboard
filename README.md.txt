# Myntra Sales Performance Dashboard (Power BI)

## Project Overview
This project analyzes Myntra product sales data to understand brand-wise product distribution, pricing and discount patterns, and customer ratings behavior.

## Key Insights
- Identified top 10 brands based on product count
- Analyzed relationship between price, discount percentage, and customer interest
- Evaluated product distribution across different rating levels
- Enabled interactive filtering using brand, price range, and ratings

## Tools Used
- Power BI Desktop
- DAX Measures
- CSV Dataset (Web Scraped Myntra Data)

## Dashboard Features
- KPI Cards for high-level metrics
- Scatter chart for price vs discount analysis
- Brand-wise product performance visuals
- Interactive slicers for dynamic analysis
